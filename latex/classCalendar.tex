\doxysubsection{Detailed Description}
Manages a collection of events and provides functionality for rendering and interacting with a calendar. 

The \doxylink{classCalendar}{Calendar} class allows adding, removing, and updating events. It also handles rendering the calendar in an ncurses window, provides date-\/related utilities, and manages the selection of events.
