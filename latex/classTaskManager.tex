{\footnotesize\ttfamily void Task\+Manager\+::move\+Selection (\begin{DoxyParamCaption}\item[{int}]{direction }\end{DoxyParamCaption})}



Moves the task selection in the specified direction. 

This function handles circular navigation for both rows (tasks) and columns (task types).
\begin{DoxyItemize}
\item {\bfseries{Direction 0\+:}} Moves the selection up within the current task type.
\item {\bfseries{Direction 1\+:}} Moves the selection down within the current task type.
\item {\bfseries{Direction 2\+:}} Moves the selection left to the previous task type.
\item {\bfseries{Direction 3\+:}} Moves the selection right to the next task type.
\end{DoxyItemize}

Circular navigation ensures that moving past the first or last element wraps around.
