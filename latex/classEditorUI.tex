\doxysubsection{Detailed Description}
Handles the rendering and interaction of a text editor\textquotesingle{}s user interface. 

The \doxylink{classEditorUI}{Editor\+UI} class is responsible for managing the main window, sidebar, and content area. It provides functions for rendering the UI, displaying content, managing scrolling, and handling user prompts. @\+Author Gordon Xu @\+Author Runcong Zhu
