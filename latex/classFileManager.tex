\doxysubsection{Detailed Description}
The \doxylink{classFileManager}{File\+Manager} class is responsible for managing file operations such as loading, saving, and creating files within the application directory. 

It handles tasks like initializing the app directory, scanning for existing files, and ensuring that a default file is created if no files are present.
