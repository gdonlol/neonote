Displays a text prompt in a window and returns the user input. 

This function creates a temporary window for displaying a text prompt to the user, who can input text. The function supports cursor movement, backspace functionality, and scrolling when the input exceeds the visible space. The user can finalize their input by pressing the Enter key, which will return the input as a string.
