Beautiful, fast, and fully featured text editor in the terminal for users who wish to keep their workflow in one place. This project will incorporate planning features similar to Notion to streamline personal productivity but be optimized for terminal use with a graphical TUI (text-\/based user interface). The text editor will integrate tools for task management, calendar scheduling, and more, while being optimized for a keyboard-\/driven workflow in a terminal environment.

This software will be developed in C++ and will run virtually on a Raspberry Pi. We do not require a physical Raspberry Pi kit for this project.\hypertarget{md_README_autotoc_md4}{}\doxysection{\texorpdfstring{Features}{Features}}\label{md_README_autotoc_md4}

\begin{DoxyItemize}
\item Markdown editing
\item Keyboard user-\/centric, efficient keybinds for everything
\item Kanban board todo list for productivity
\item Calendars for event planning
\item Interface for list of pages on sidebar
\end{DoxyItemize}\hypertarget{md_README_autotoc_md5}{}\doxysection{\texorpdfstring{Keybinds Guide}{Keybinds Guide}}\label{md_README_autotoc_md5}
This guide provides an overview of the keybindings available in the terminal editor, covering both content editing and sidebar navigation.


\begin{DoxyEnumerate}
\item \#\# Main Menu Navigation
\end{DoxyEnumerate}
\begin{DoxyItemize}
\item Arrow Up (↑) or {\ttfamily k} -\/ Moves up the menu options.
\item Arrow Down (↓) or {\ttfamily j} -\/ Moves down the menu options.
\item Enter (⏎) or {\ttfamily \textbackslash{}n} -\/ Selects the currently highlighted menu option (either "{}my notes"{} or "{}exit"{}).
\end{DoxyItemize}
\begin{DoxyEnumerate}
\item \#\# Content Editing Mode
\end{DoxyEnumerate}
\begin{DoxyItemize}
\item Arrow Keys (↑, ↓, ←, →) -\/ Move the cursor up, down, left, or right.
\item Backspace (⌫) -\/ Deletes the character to the left of the cursor.
\item Enter (⏎) -\/ Inserts a new line at the current cursor position.
\item {\ttfamily Ctrl + S} -\/ Saves the current file.
\item {\ttfamily Ctrl + Q} -\/ Saves and exits the editor.
\item {\ttfamily Ctrl + B} -\/ Inserts \texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*} (bold formatting marker).
\item {\ttfamily Ctrl + I} -\/ Inserts \texorpdfstring{$\ast$}{*} (italic formatting marker).
\item {\ttfamily Ctrl + O} / {\ttfamily Ctrl + D} -\/ Switches focus to the sidebar.
\item {\ttfamily Ctrl + N} -\/ Makes a new file.
\item {\ttfamily Ctrl + \mbox{]}} -\/ Move the cursor to the start of the previous word.
\item {\ttfamily Ctrl + \textbackslash{}} -\/ Skip to the next word.
\item Printable Characters ({\ttfamily A-\/Z}, {\ttfamily 0-\/9}, {\ttfamily Symbols}) -\/ Inserts the typed character at the cursor position.
\end{DoxyItemize}
\begin{DoxyEnumerate}
\item \#\# Sidebar Navigation Mode
\end{DoxyEnumerate}
\begin{DoxyItemize}
\item Arrow Up (↑) -\/ Moves up the file list.
\item Arrow Down (↓) -\/ Moves down the file list.
\item Enter (⏎) -\/ Loads the selected file or page into the editor.
\item {\ttfamily Ctrl + O} / {\ttfamily Ctrl + D} -\/ Switches focus back to content editing.
\item {\ttfamily Ctrl + R} -\/ Rename a file (click enter to confirm after typing in new name).
\item {\ttfamily Ctrl + T} -\/ Create a new task for the kanban board (click enter to confirm after typing in new name).
\item {\ttfamily Ctrl + P} -\/ Move an existing task into a new category (to do, in progress or done).
\end{DoxyItemize}
\begin{DoxyEnumerate}
\item \#\# Cursor and Scroll Adjustments
\end{DoxyEnumerate}
\begin{DoxyItemize}
\item Cursor movement is restricted within the text bounds.
\item Scrolling occurs automatically when the cursor moves beyond the visible area. 
\end{DoxyItemize}
