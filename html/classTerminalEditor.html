◆ redraw()

void TerminalEditor::redraw ( )

Redraws the terminal editor interface.

This function adjusts the window sizes based on the current terminal dimensions, repositions the sidebar and content windows, and renders the appropriate section based on the current focus:

  • Content editor: Displays the current file with cursor visibility.
  • Task manager: Renders the task list.
  • Calendar: Displays the calendar view.

It also ensures the sidebar and main content are properly refreshed.
